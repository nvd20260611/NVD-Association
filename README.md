# NVD Association Official Website

NVD Road Feedback & Repair official static website.

This repository contains the public-facing association website for:

- Road feedback and repair action
- Civic resilience communication
- Brand manifesto and association positioning
- Participation and co-creation information

## Structure

- `index.html` - Home page
- `resilience.html` - Road feedback and civic resilience project
- `manifesto.html` - Brand manifesto
- `join.html` - Participation page
- `styles.css` - Global styling
- `script.js` - Front-end interactions and localization
- `assets/` - Public visual assets and source records

## Deployment

This site is designed for GitHub Pages.

Recommended setting:

- Source: Deploy from a branch
- Branch: `main`
- Folder: `/root`
